## 经验总结

### 1. 哪些后缀值得投资

按价值高低顺序，分别为：.com、.cn、.com.cn、.net，推荐 .com、.cn。

### 2. 什么时候才能抄底

注意看群气氛，大致分为 4 个步骤：

1. 刚开始跌，群里人吹牛扯皮，谈笑风生；
2. 跌了 20%，开始问什么时候加仓,回忆过去，安慰自己；
3. 跌了 50%，问加仓的人少了，开始飙段子，不谈行情，麻木自己；
4. 跌了 80%，群里大部分人不吭声了，偶尔几个查价格的；

过了这 4 步，当群里人开始减少，有些群解散了时，才是加仓的机会，稳稳的涨幅。
